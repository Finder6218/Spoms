# 豆制品订货销售管理系统

《豆制品订货销售管理系统》是专门为豆制品经销商管理订货销售开发的一款软件。该软件操作简便，查询统计快捷灵活，可以使用户轻易上手！ 

系统包含以下功能：   
1.订货管理：订货登记、订货查询、今日未订货查询、按客户查询、按期间查询、按期间统计；   
2.基本信息：产品信息、客户信息。 

软件特点：   
1.支持数据与execel的导出与导入；   
2.数据查询功能强大；   
3.信息录入方便简单。 

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
